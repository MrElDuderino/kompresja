# Lista 7: Kody korekcyjne

## Sposób uruchomienia:

kodowanie:
	`python3 main.py koder <plik wejściowy> <plik wyjściowy>`

zaszumianie:
	`python3 main.py szum <prawdopodobieństwo> <plik wejściowy> <plik wyjściowy>`

dekodowanie:
	`python3 main.py dekoder <plik wejściowy> <plik wyjściowy>`

sprawdzenie:
	`python3 main.py sprawdz <plik wejściowy> <plik wyjściowy>`
